Operating Sytems



Definition:

An operating system is a program that acts as an interface between the user and the computer hardware and controls the execution of all kinds of programs.




The following list includes some of the most important functions of any opearting system:

  • Memory Managament

  • Processor Management

  • Device Management

  • File Management


Different types of oprating systems include:

  • Real Time OS

  • Distributed OS

  • Network OS

  • Mobile OS

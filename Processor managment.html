Processor Managment




In multiprogramming environment, the OS decides which process gets the processor when and for how much time. This function is called process scheduling. An Operating System does the following activities for processor managemen

  • Keeps tracks of processor and status of process. The program responsible for this task is known as traffic controller.

  • Allocates the processor (CPU) to a process.

  • De-allocates processor when a process is no longer required.
